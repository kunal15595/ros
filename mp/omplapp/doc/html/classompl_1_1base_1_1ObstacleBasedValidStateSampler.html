Detailed Description

Generate valid samples using obstacle based sampling. First sample an invalid state, then sample a valid state. Then, interpolate from the invalid state to the valid state, returning the first valid state encountered.

External documentation
N. M. Amato, O. B. Bayazit, L. K. Dale, C. Jones, and D. Vallejo, OBPRM: an obstacle-based PRM for 3D workspaces, in Third Workshop on the Algorithmic Foundations of Robotics, pp. 155-168, 1998. DOI: http://dl.acm.org/citation.cfm?id=299002
